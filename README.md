# MEVN Todo Application

This is simple Todo application developed using MongoDB, Express JS, Vue JS and Node JS.

  - Creating Todo
  - Deleting Todo

Clone the repo and follow the below instructions to run the client and server seperately.

### Client Installation

```sh
$ cd client
$ npm install
$ npm run serve
```

### Server Installation
```sh
$ cd server
$ npm install
$ nodemon
```
